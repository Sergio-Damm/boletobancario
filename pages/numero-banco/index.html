O número do banco

O número do banco é um código de três dígitos que identifica de forma única cada instituição financeira no Brasil, garantindo que o dinheiro seja enviado para o banco correto

O que é o número do banco?

O número do banco também é chamado de código do banco, número código da instituição destinatária ou código COMPE (Código do Sistema de Operações Monetárias e Compensação de Outros Papéis).

Exemplos:

  • Banco do Brasil = 001
  • Caixa = 104
  • Bradesco = 237
  • Itaú = 341

Uma linha digitável, que é uma identificação padronizada pelo Banco Central do Brasil, sempre começa com o número do banco.

Exemplo de linha digitável: 00190500954014481606906809350314337370000000100.

O número do banco pode ser conferido oficialmente na lista atualizada de participantes do Sistema de Transferência de Reservas (STR) no site do Banco Central do Brasil.
