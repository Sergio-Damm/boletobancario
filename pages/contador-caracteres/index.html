Formulário contador de caracteres

O contador de caracteres é um utilitário digital que, em tempo real, informa o número total de letras, números, espaços e pontuações em um texto. Muito utilizado para limitar inputs, garantindo que textos não excedam limites de sistemas.

Formulário contador de caracteres

Total de caracteres: 0

Números: 0

Letras: 0

Palavras: 0

Espaços: 0

Símbolos/Pontuações: 0

Linhas: 0

Como o formulário funciona

O "Total de caracteres" inclui todos os caracteres visíveis e invisíveis (espaços).

Cada quebra de linha (após pressionar a tecla Enter) é contada como 1 caractere. Letras de A a Z, maiúsculas ou minúsculas, são contadas como caracteres. Números de 0 a 9 são contados como caracteres. Símbolos e pontuações (pontos, vírgulas, caracteres especiais, etc.) são contados como caracteres. Palavras são contadas apenas como sequências de letras (A-Z), excluindo números.

O formulário considera parágrafos como "Linhas". Cada vez que você pula uma linha (tecla Enter), conta como 1 linha. Um parágrafo longo (texto longo), mesmo que ocupe 5 linhas visualmente, conta apenas como uma linha pelo sistema.
