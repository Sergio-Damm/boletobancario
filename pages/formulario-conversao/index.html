Formulário para Conversão da Linha Digitável do Boleto Bancário na Representação Gráfica ou Numérica do Código de Barras e vice versa

Para vencimentos até 21/02/2025 utilize o Formulário de Conversão - Antigo

Atenção:

A linha digitável é válida, mas o fator de vencimento resulta em vencimento: .

O fator de vencimento foi reiniciado em 22/02/2025 com valor 1000. Portanto, um fator de vencimento é estranho e indica um vencimento muito distante. Confirme se o Boleto realmente tem vencimento superior a 90 dias.

Se o vencimento for anterior a 21/02/2025, utilize o formulário antigo.

O código de barras aparecerá aqui

Novo fator de vencimento: a partir de 22/02/2025

Este formulário de conversão foi elaborado considerando o fator de vencimento 22/02/2025 = fator 1000 (data base = 29/05/2022).

Para mais informações sobre o fator de vencimento consulte a nossa página Data de Vencimento e Valor.

Conversão da Linha Digitável em Código de Barras

Digite a Linha Digitável (47 números, sem espaços, pontos ou vírgulas) no campo "Digite aqui a linha digitável" e clique em "Calcular Código de Barras".

Será informado o Código de Barras convertido em números (44 números, conforme detalhado no link Linha Digitável).

Além da conversão, o formulário informará também a Data de Vencimento e o Valor do Boleto Bancário.

Conversão do Código de Barras em Linha Digitável

Para converter o Código de Barras (com 44 dígitos) na Linha Digitável (com 47 dígitos), clique com o mouse dentro do campo "Informe aqui o código de barras" e escaneie (digitalize) o código de barras com um leitor de código de barras (scanner). Depois é só clicar no botão "Calcular a linha digitável" para obter a Linha Digitável (com 47 dígitos).

Outras informações

Os critérios aqui utilizados obedecem aos padrões da Federação Brasileira de Bancos - FEBRABAN e do Banco Central do Brasil - BCB.

Este formulário é válido somente para conversão de Boletos Bancários de pagamentos de produtos e serviços (não é válido para Boletos de Concessionárias: luz, água, etc.).

Para converter Boletos de Concessionárias (pagamento de luz, água, gás, telefone, impostos, etc.), observe os critérios do Layout Padrão de Arrecadação/Recebimento com Utilização do Código de Barras da Febraban, em vigência a partir de 01.03.2023.
