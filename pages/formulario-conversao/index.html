Para que serve o Formulário de Conversão?

O formulário acima converte (transforma) os números de um Boleto Bancário — a Linha Digitável — em sua representação gráfica, o Código de Barras, e também permite a conversão do Código de Barras nos números da Linha Digitável.

Essa funcionalidade facilita o pagamento, a leitura por máquinas e a conferência das informações, especialmente quando o Código de Barras ou a Linha Digitável estão danificados, bem como para a inserção correta dos dados em sistemas e programas.

O formulário assegura que informações como banco, vencimento e valor sejam transferidas corretamente para os sistemas de pagamento, ajudando a prevenir erros e fraudes, sendo essencial para a automação bancária.

Para que serve especificamente:

  • Pagamento manual de boleto: permite digitar a Linha Digitável do boleto bancário em caixas eletrônicos, internet banking ou aplicativos de bancos, possibilitando a geração e leitura do Código de Barras para processamento do pagamento.
  • Segurança e validação dos dados: a Linha Digitável possui dígitos verificadores que confirmam a autenticidade das informações do boleto, ajudando a prevenir fraudes e erros de digitação.
  • Geração do Código de Barras: converte a sequência numérica da Linha Digitável em um Código de Barras legível por leitores ópticos, sendo útil para sistemas de gestão, automação bancária e emissão de boletos.
  • Recuperação de informações do boleto: quando o Código de Barras físico está danificado ou ilegível, a Linha Digitável pode ser utilizada para gerar o código novamente ou identificar corretamente o boleto bancário.
  • Conformidade com o padrão Febraban: garante que os dados numéricos do boleto sigam os padrões definidos pela Febraban, assegurando compatibilidade com qualquer instituição financeira no Brasil.

Em resumo, o Formulário atua como uma ferramenta de intermediação e validação, tornando os dados do boleto acessíveis tanto para leitura humana (Linha Digitável) quanto para leitura automática (Código de Barras), garantindo a fluidez e segurança das transações financeiras.

Conversão da Linha Digitável em Código de Barras

Digite a Linha Digitável (47 números, sem espaços, pontos ou vírgulas) no campo "Digite aqui a linha digitável" e clique em "Calcular Código de Barras".

Será informado o Código de Barras convertido em números (44 números, conforme detalhado no link Linha Digitável).

Além da conversão, para verificação, o formulário informará também a Data de Vencimento e o Valor do Boleto Bancário.

Conversão do Código de Barras em Linha Digitável

Para converter o Código de Barras (com 44 dígitos) na Linha Digitável (com 47 dígitos), clique com o mouse dentro do campo "Informe aqui o código de barras" e escaneie (digitalize) o código de barras com um leitor de código de barras (scanner). Depois é só clicar no botão "Calcular a linha digitável" para obter a Linha Digitável (com 47 dígitos).

Outras informações

Os critérios aqui utilizados obedecem aos padrões da Federação Brasileira de Bancos - FEBRABAN e do Banco Central do Brasil - BCB.

Este formulário é válido somente para conversão de Boletos Bancários de pagamentos de produtos e serviços (não é válido para Boletos de Concessionárias: luz, água, etc.).

Para converter Boletos de Concessionárias (pagamento de luz, água, gás, telefone, impostos, etc.), observe os critérios do Layout Padrão de Arrecadação/Recebimento com Utilização do Código de Barras da Febraban, em vigência a partir de 01.03.2023.
