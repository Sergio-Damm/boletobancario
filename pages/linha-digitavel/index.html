A Linha Digitável e o Código de Barras

A Linha Digitável é a representação numérica do Código de Barras. Já o Código de Barras é a representação gráfica do conteúdo do boleto bancário (código do banco, código da moeda, dígitos verificadores, vencimento, valor e campo livre)

Linha Digitável e Código de Barras: o que são e para que servem?

A Linha Digitável e o Código de Barras são duas formas de representar as mesmas informações do boleto bancário, em sequências e formatos diferentes.

O Código de Barras é a representação gráfica do conteúdo do boleto bancário e a Linha Digitável é a representação numérica do Código de Barras.

O que é a Linha Digitável?

A Linha Digitável é composta por 47 dígitos (números), localizada na parte superior direita do boleto bancário, que contém informações importantes do boleto, como:

  • Identificação do Banco
  • Código da Moeda
  • Campos de Controle e Identificação do Beneficiário
  • Fator de Vencimento
  • Valor a ser Pago
  • Dígitos Verificadores

Para que serve a Linha Digitável?

A Linha Digitável é utilizada para pagamentos manuais via internet banking, aplicativos ou caixas eletrônicos quando a leitura automática do Código de Barras não é possível.

Principais funções da Linha Digitável:

  • Pagamento Manual: Permite digitar as informações do boleto em caixas eletrônicos, internet banking ou aplicativos de bancos.
  • Verificação de Dados: O sistema bancário verifica se as informações estão corretas antes de processar o pagamento.
  • Prevenção de Erros e Fraudes: Os dígitos verificadores ajudam a evitar erros de digitação e fraudes.

O que é o Código de Barras?

O Código de Barras é uma representação gráfica, em formato de barras verticais pretas e brancas de diferentes espessuras, do conteúdo do Boleto Bancário.

As barras verticais pretas, juntamente com os espaços brancos entre elas, são lidas por sensores óticos como "0s" e "1s" (código binário). As barras pretas e os espaços brancos variam em espessura (finas ou largas) para representar os dígitos decimais (0-9) que compõem os números do Boleto Bancário.

O Código de Barras foi criado para ser lido com rapidez e precisão, transformando o conteúdo do Boleto Bancário em dígitos.

As barras verticais pretas e brancas de diferentes espessuras são do tipo "2 de 5 intercalado" (Intercalado 2 de 5 - ITF).

O termo "2 de 5" significa que 5 barras (pretas ou brancas) definem 1 dígito, sendo que duas delas são sempre largas. O termo "intercalado" indica que os espaços entre as barras pretas também carregam informação, de maneira análoga às barras.

O resultado visual é um bloco de 10 elementos (5 barras pretas + 5 espaços brancos) que, intercalados, representam sempre um par de dígitos.

Como o leitor sabe onde o Código de Barras começa e termina?

Para que os scanners e os aplicativos não se confundam com outras impressões no papel, o padrão Intercalado 2 de 5 utiliza marcas de início e fim, chamadas de caracteres de guarda:

  • Início: O código sempre começa com duas barras pretas finas e dois espaços brancos finos (sequência NNNN). Isso avisa ao leitor: "Atenção, os próximos blocos são dígitos".
  • Fim: O código termina com uma barra preta larga, um espaço branco fino e uma barra preta fina (sequência WNN). Isso indica ao leitor: "O código acabou, pode parar de ler".

Sem esses sinais, o leitor óptico não saberia se está lendo o Boleto Bancário da esquerda para a direita ou vice-versa, nem onde exatamente a informação numérica começa.

Para que serve o Código de Barras?

  • Pagamento Rápido e Prático: A leitura eletrônica das barras agiliza o processo de pagamento.
  • Redução de Erros: A leitura eletrônica reduz a chance de erros de digitação.
  • Automatização do Pagamento: Os dados são enviados diretamente para o sistema bancário.

Qual a relação entre a Linha Digitável e o Código de Barras?

O Boleto Bancário (Bloqueto de Cobrança) foi criado em 1993, através da Carta Circular n° 2.414, de 7 de outubro de 1993, do Banco Central do Brasil. Esse normativo criou também a Ficha de Compensação e o Código de Barras, ambos fundamentais para a implantação da Compensação Eletrônica de Cobrança.

Mais tarde, em 1995, para "aprimoramento do Bloqueto de Cobrança", a Linha Digitável do Boleto Bancário foi criada (Carta Circular nº 2.531, de 24 de fevereiro de 1995).

A Linha Digitável foi criada depois do Código de Barras, para que o usuário pudesse digitar a sequência numérica caso o leitor do Código de Barras não funcionasse.

No entanto, é importante lembrar que a Linha Digitável e o Código de Barras contêm as mesmas informações essenciais, mas organizadas de formas diferentes:

  • Linha Digitável: 47 dígitos (44 dígitos do Código de Barras + 3 dígitos verificadores).
  • Código de Barras: 44 dígitos (que representam o conteúdo do boleto em formato de barras verticais).

Exemplo:

  • Linha Digitável: 001 9 0500 9 5 4014481606 9 0680935031 4 3 3737 0000000100
  • Código de Barras: 001 9 3 3737 0000000100 0500 9 401448 1606 06809350 31

A diferença de 3 dígitos, entre a Linha Digitável e o Código de Barras, são os dígitos verificadores, que são adicionados à Linha Digitável para garantir a integridade das informações e evitar erros durante a digitação.

Tanto a Linha Digitável quanto o Código de Barras seguem as especificações técnicas, incluindo a fórmula de cálculo do dígitos verificadores da Linha Digitável (módulos 10 e 11), do Manual Técnico do Banco do Brasil - BB.

Os dígitos da Linha Digitável

A Linha Digitável é a representação numérica do Código de Barras.

Ambos, a Linha Digitável e o Código de Barras, contém as mesmas informações, mas são apresentados em formatos e sequências diferentes.

Exemplo de uma Linha Digitável com 47 dígitos, sem espaços, pontos, vírgulas ou qualquer outro caractere:

00190500954014481606906809350314337370000000100

A Linha Digitável é distribuída em 5 campos (partes), sendo os 3 primeiros consistidos por Dígito Verificador – DV (calculado através do Módulo 10) e, entre cada campo, espaço equivalente a uma posição. No quarto campo é indicado, isoladamente, o DV (calculado através do Módulo 11) do Código de Barras:

A Linha Digitável separada por campos:

001 9 05009 ( 5 ) 401448 1606 ( 9 ) 0680935031 ( 4 ) 3 37370000000100 (47 dígitos)

Observe que os dígitos com cores roxas acima também constam no Código de Barras abaixo, mas em ordem diferente (3 37370000000100).

Os "dígitos" do Código de Barras

O Código de Barras é a representação gráfica do conteúdo do Boleto Bancário.

Exemplo de um Código de Barras com 44 dígitos, o mesmo significado da Linha Digitável acima, sem espaços, pontos, vírgulas ou qualquer outro caractere:

00193373700000001000500940144816060680935031

O Código de Barras separado por campos:

001 9 3 37370000000100 05009 401448 16060680935031 (44 dígitos)

Observe que os dígitos com cores azuis, por exemplo, são os mesmos, tanto na Linha Digitável quanto no Código de Barras (05009).

A Linha Digitável e o Código de Barras são compostos por 2 Campos: o "Campo Obrigatório" e o "Campo Livre".

  • Campo Obrigatório: Determinado pela FEBRABAN e comum a todos os bancos.
  • Campo Livre: Determinado por cada banco de acordo com a modalidade de Cobrança utilizada pelo cliente.
