A Linha Digitável e o Código de Barras

A Linha Digitável é a representação numérica do Código de Barras. Já o Código de Barras é a representação gráfica do conteúdo do Boleto Bancário (código do banco, código da moeda, dígito verificador, fator de vencimento, valor do documento e campo livre)

Linha Digitável e Código de Barras: o que são e para que servem?

A Linha Digitável e o Código de Barras são duas formas de representar as mesmas informações do Boleto Bancário, em formatos diferentes.

O Código de Barras é a representação gráfica do conteúdo do Boleto Bancário e a Linha Digitável é a representação numérica do Código de Barras.

O que é a Linha Digitável?

A Linha Digitável é uma sequência de 47 números que contém informações importantes do boleto, como:

  • Banco emissor
  • Valor a ser pago
  • Data de vencimento
  • Dígitos verificadores

Para que serve a Linha Digitável?

A Linha Digitável é utilizada para pagamentos manuais via internet banking, aplicativos ou caixas eletrônicos quando a leitura automática do Código de Barras não é possível.

Principais funções da linha digitável:

  • Pagamento manual: Permite digitar as informações do boleto em caixas eletrônicos, internet banking ou aplicativos de bancos.
  • Verificação de dados: O sistema bancário verifica se as informações estão corretas antes de processar o pagamento.
  • Prevenção de erros e fraudes: Os dígitos verificadores ajudam a evitar erros de digitação e fraudes.

O que é o Código de Barras?

O Código de Barras é uma sequência de 44 barras verticais de diferentes espessuras.

Para que serve o Código de Barras?

  • Pagamento rápido e prático: A leitura eletrônica das barras agiliza o processo de pagamento.
  • Redução de erros: A leitura eletrônica reduz a chance de erros de digitação.
  • Automatização do pagamento: Os dados são enviados diretamente para o sistema bancário.

Qual a relação entre a Linha Digitável e o Código de Barras?

A Linha Digitável e o Código de Barras contêm as mesmas informações, mas em formatos diferentes:

  • Linha Digitável: 47 números (44 números do Código de Barras + 3 dígitos verificadores).
  • Código de Barras: 44 barras verticais.

Exemplo:

  • Linha Digitável: 001 9 0500 9 5 4014481606 9 0680935031 4 3 3737 0000000100
  • Código de Barras: 001 9 3 3737 0000000100 0500 9 401448 1606 06809350 31

A diferença de 3 dígitos são os dígitos verificadores, que são adicionados à Linha Digitável para garantir a integridade das informações e evitar erros durante a digitação.

Tanto a Linha Digitável quanto o Código de Barras seguem as especificações técnicas, incluindo a fórmula de cálculo do dígito verificador da Linha Digitável, do Manual Técnico do Banco do Brasil - BB.

Os números da Linha Digitável

Exemplo de uma Linha Digitável e de um Código de Barras com o mesmo significado:

001 9 05009 ( 5 ) 401448 1606 ( 9 ) 0680935031 ( 4 ) 337370000000100 (47 dígitos)

Observe que os números com cores roxas acima também constam no Código de Barras abaixo, mas em ordem diferente (337370000000100).

Os "números" do Código de Barras

Exemplo do Código de Barras com o mesmo significado da Linha Digitável acima:

001 9 337370000000100 05009 401448 16060680935031 (44 dígitos)

Observe que os números com cores azuis, por exemplo, são os mesmos, tanto na Linha Digitável quanto no Código de Barras (05009).
