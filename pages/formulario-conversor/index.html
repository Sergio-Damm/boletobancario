Converta a Linha Digitável em Código de Barras (e vice-versa)

Este formulário online foi desenvolvido para facilitar a conversão da Linha Digitável de um Boleto Bancário em Código de Barras, e também o processo inverso, permitindo a conferência de dados e a recuperação de informações, especialmente quando o boleto está danificado, ilegível ou precisa ser inserido manualmente em sistemas bancários, sempre seguindo rigorosamente os padrões definidos pela Febraban.

Para que serve o formulário de conversão de boleto?

O formulário converte os números de um Boleto Bancário — conhecidos como a Linha Digitável — em sua representação gráfica, o Código de Barras, e realiza o processo inverso quando necessário.

Essa conversão facilita:

  • O pagamento manual de boletos em caixas eletrônicos e aplicativos bancários.
  • A leitura automática por máquinas e leitores ópticos.
  • A conferência de informações como banco, valor e vencimento.
  • A inserção correta dos dados em sistemas financeiros e programas de gestão.

O processo ajuda a reduzir erros de digitação, prevenir fraudes e garantir que os dados sejam transferidos corretamente para os sistemas de pagamento.

Quando usar este conversor de Linha Digitável e Código de Barras

Você pode utilizar este conversor sempre que precisar:

  • Converter a Linha Digitável em Código de Barras.
  • Converter o Código de Barras em Linha Digitável.
  • Recuperar informações de um boleto com Código de Barras danificado.
  • Conferir valor, vencimento ou banco emissor de um boleto.
  • Validar dados antes de efetuar o pagamento.

Exemplos práticos de uso

Este conversor de boletos é útil em situações como:

  • Digitar manualmente um boleto no aplicativo do banco.
  • Reconstruir o Código de Barras de um boleto rasgado ou apagado.
  • Conferir se o valor do boleto foi alterado indevidamente.
  • Validar valores e vencimentos de boletos recebidos por e-mail ou aplicativos de mensagem.
  • Utilizar ou importar os dados corretos para sistemas financeiros ou ERPs.

Funcionalidades específicas do formulário

  • Pagamento manual de boleto: permite gerar o Código de Barras a partir da Linha Digitável para facilitar a leitura e o processamento.
  • Segurança e validação: a Linha Digitável possui dígitos verificadores que confirmam a autenticidade das informações.
  • Geração de Código de Barras: converte a sequência numérica em um código legível por leitores ópticos.
  • Recuperação de dados: possibilita identificar e validar boletos mesmo quando o código físico está ilegível.
  • Conformidade Febraban: garante compatibilidade com qualquer instituição bancária no Brasil.

Como converter a Linha Digitável em Código de Barras

Digite ou copie e cole a Linha Digitável (47 números, sem espaços, pontos ou vírgulas) no campo "Digite aqui a Linha Digitável" e clique em "Calcular o Código de Barras".

O formulário exibirá:

  • O Código de Barras convertido (e os 44 dígitos do Código de Barras)
  • O Valor do Boleto
  • A Data de Vencimento

Como converter o Código de Barras em Linha Digitável

  • Clique dentro do campo "Informe aqui o Código de Barras".
  • Escaneie o Código de Barras com um leitor óptico (scanner) ou digite os 44 dígitos.
  • Clique em "Calcular a Linha Digitável".
  • O formulário exibirá a Linha Digitável completa (47 dígitos).

Perguntas frequentes sobre Boleto Bancário

É possível pagar um boleto apenas com a Linha Digitável?

Sim. A Linha Digitável contém todas as informações necessárias para pagamento manual.

O Código de Barras e a Linha Digitável possuem os mesmos dados?

Sim. Ambos representam exatamente as mesmas informações do boleto, apenas em formatos diferentes.

Este conversor funciona para qualquer tipo de boleto?

Este formulário é válido apenas para Boletos Bancários padrão Febraban, utilizados para pagamento de produtos e serviços.

O site armazena os dados informados?

Não. Nenhuma informação digitada é armazenada, registrada ou compartilhada.

Informações importantes

Desde 2018, é obrigatório que todos os Boletos Bancários de Cobrança sejam registrados na base centralizadora da Nuclea (antiga CIP), conforme normas da Febraban e do Banco Central. O registro garante maior segurança, evitando fraudes, pagamentos em duplicidade e permitindo o pagamento em qualquer banco (mesmo vencido).

Para saber se um Boleto Bancário é registrado, a forma mais segura é utilizar o DDA (Débito Direto Autorizado) no aplicativo do seu banco, que lista todas as cobranças vinculadas ao seu CPF/CNPJ. Boletos registrados contêm dados do pagador, beneficiário e banco.

Outra forma de verificar se um Boleto Bancário é registrado é fazer uma "simulação de pagamento". Digite a Linha Digitável ou leia o Código de Barras com o aplicativo do seu banco. Se for registrado, o banco exibirá automaticamente os dados do beneficiário (nome e CNPJ) para confirmação antes do pagamento.

Aviso de segurança

Sempre confira cuidadosamente os dados do Boleto Bancário, principalmente:

  • Nome, número do CPF ou CNPJ do beneficiário
  • Valor do boleto
  • Data de vencimento
  • Banco emissor

Termos de uso e responsabilidade

  • Não temos qualquer vínculo com bancos ou instituições financeiras.
  • As informações geradas possuem caráter meramente informativo.
  • O site não realiza transações financeiras.
  • O uso das informações é de total responsabilidade do usuário.

Outras informações importantes

  • Este formulário serve exclusivamente para testes de sistemas de pagamentos, de acordo com o padrão Febraban.
  • Este formulário não é válido para boletos de concessionárias (água, luz, gás, telefone, impostos, etc.).
  • Para boletos de concessionárias, veja instruções no arquivo Layout Padrão de Arrecadação com Código de Barras, da Febraban, vigente a partir de 01/03/2023.
  • Para vencimentos antes de 21/02/2025 utilize o Formulário de Conversão - Antigo.
